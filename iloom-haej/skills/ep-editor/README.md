# ep-editor

.md 품의서를 네이버 스마트에디터(SE) HTML로 변환하는 스킬.

## 사용법

```
/ep-editor [파일경로.md]
```

예시:
```
/ep-editor 10-projects/13-motion-campaign-contents/motion-campaign-branded-contents-approval.md
```

## SE 에디터 적용 방법

1. EP 에디터 메뉴에서 `</>` (소스보기) 버튼 클릭
2. 기존 내용 전체 삭제 후 변환된 코드 붙여넣기
3. 다시 `</>` 클릭하여 일반 편집 모드 복귀

## 변환 스타일

| 요소 | 스타일 |
|------|--------|
| 대제목 (H2) | `◆` 불렛 + 17px 볼드 |
| 소제목 (H3) | 빨간 왼쪽 선(`#c80a1e`) + 15px 볼드 |
| 소소제목 (H4) | 15px medium, 회색(`#555555`) |
| 본문 | 15px regular, 다크(`#333333`) |
| 리스트 | `·` 불렛, 들여쓰기 16px |
| 표 | 회색 테두리, 헤더 배경 `#F8F8F8` |
| 금액 강조 | 빨간 볼드(`#c80a1e`) |
