# iloom-sunwoo Plugin

일룸 AX 업무 자동화 플러그인 (담당: 김선우)

## 포함된 컴포넌트

| 종류 | 이름 | 용도 |
|------|------|------|
| Skill | competitor-review-analyzer | 경쟁사 리뷰 크롤링 + 분석 |
| Skill | transcript-organizer | 회의록/강의 녹음 정리 |
| Command | `/iloom-sunwoo:sns-draft` | SNS 콘텐츠 초안 생성 |
| Command | `/iloom-sunwoo:monitoring` | 브랜드 모니터링 리포트 |
| Command | `/iloom-sunwoo:thinking-partner` | 브레인스토밍 파트너 |
| Agent | research-worker | 웹 리서치, 팩트 수집 |
| Agent | analysis-worker | 데이터 분석, 패턴 발견 |

## 설치

```bash
/plugin install iloom-sunwoo
```
