# /sns-draft - SNS 콘텐츠 초안 생성

SNS 채널별 콘텐츠 초안을 자동으로 생성하는 커맨드입니다.

## 사용법

```bash
/sns-draft                                          # 대화형 모드
/sns-draft --channel instagram --topic "신제품"    # 채널/주제 지정
/sns-draft --channel all --topic "봄 캠페인"       # 전 채널 동시 생성
```

## 프로세스

### 1단계: 브랜드 컨텍스트 로드

사용자 CLAUDE.md에서 SNS 관련 설정을 확인합니다:

```yaml
SNS 채널별 톤앤매너:
  instagram:
    톤: 친근하고 영감을 주는
    길이: 200자 내외
    해시태그: 5-10개
  youtube:
    톤: 정보 전달 + 신뢰감
    썸네일: 10자 내외, 숫자/질문 활용
  blog:
    톤: 전문적이면서 따뜻한
    길이: 1,500-2,000자
```

설정이 없으면 기본 일룸 브랜드 가이드를 적용합니다.

### 2단계: 주제/무드 확인

사용자가 지정하지 않은 경우 대화형으로 확인:

```
Q: 어떤 주제의 콘텐츠를 만들까요?
> 2026 봄 신제품 책상

Q: 원하는 무드나 톤이 있나요? (없으면 Enter)
> 새학기 설렘, 새출발
```

## 저장 위치

생성된 콘텐츠는 다음 경로에 저장됩니다:
`20-sns/{channel}/{YYYY-MM-DD}_{topic}.md`

예시:
- `20-sns/instagram/2026-01-20_봄신제품.md`
- `20-sns/youtube/2026-01-20_봄신제품.md`

## 옵션

| 옵션 | 설명 | 예시 |
|------|------|------|
| `--channel` | 채널 지정 | `instagram`, `youtube`, `blog`, `all` |
| `--topic` | 주제 지정 | `--topic "신제품 론칭"` |
| `--mood` | 무드/톤 지정 | `--mood "밝고 활기찬"` |
| `--format` | 세부 형식 | `--format carousel` (인스타 캐러셀) |
| `--output` | 저장 경로 | `--output ./drafts/` |

## 채널별 특화 옵션

### Instagram
```bash
/sns-draft --channel instagram --format carousel  # 캐러셀
/sns-draft --channel instagram --format reels     # 릴스
/sns-draft --channel instagram --format story     # 스토리
```

### YouTube
```bash
/sns-draft --channel youtube --format shorts      # Shorts (60초)
/sns-draft --channel youtube --format long        # 일반 영상 (5-10분)
```

## 활용 팁

### 1. 주간 콘텐츠 캘린더 생성

```bash
"이번 주 인스타그램 콘텐츠 캘린더 만들어줘.
주제: 봄 신제품 론칭
게시 일정: 월/수/금 오후 7시"
```

### 2. 멀티채널 동시 생성

```bash
/sns-draft --channel all --topic "봄 캠페인"
# 인스타, 유튜브, 블로그용 동시 생성
```

### 3. 시리즈 콘텐츠

```bash

1편: 공간 세팅, 2편: 데스크 아이템, 3편: 정리 팁"
```
