# /monthly-report - 월간 보고서 자동 생성

매월 이 커맨드 하나로 월간 보고서를 자동 생성하고 Confluence에 업로드합니다.

---

## 실행 방법

```
/monthly-report          → 이번 달 보고서 생성
/monthly-report 2026-03  → 특정 월 보고서 생성
```

---

## Claude가 수행할 작업 (실제 동작 지침)

이 커맨드가 실행되면 Claude는 아래 순서대로 **실제로** 작업을 수행합니다.

### STEP 1. 대상 월 확인 및 파일 탐색

1. 인자가 있으면 해당 월, 없으면 이번 달로 설정
2. `00-inbox/` 폴더에서 아래 파일을 탐색:

| 파일 패턴 | 필수 여부 | 용도 |
|-----------|----------|------|
| `수주실적_YYYYMM.xlsx` 또는 `수주*.xlsx` | **필수** | 대리점별 수주 데이터 |
| `매출실적_YYYYMM.xlsx` 또는 `매출*.xlsx` | **필수** | 대리점별 매출 데이터 |
| `목표_YYYY.xlsx` 또는 `목표*.xlsx` | 권장 | 연간 수주/매출 목표 |
| `외부활동_YYYYMM.xlsx` 또는 `외부활동*.xlsx` | 선택 | 제휴/이벤트 성과 |

3. **필수 파일이 없으면** 사용자에게 안내:
   ```
   "수주실적 파일이 00-inbox/ 에 없습니다.
    파일을 넣어주시거나 경로를 알려주세요."
   ```
4. 선택 파일이 없으면 해당 섹션 건너뛰기 여부를 확인

### STEP 2. 엑셀 데이터 분석

각 파일을 `/excel-analyzer` 방식으로 분석합니다.

**수주실적 분석:**
- 대리점별 수주 합계
- 상권별(일산/파주/대구) 분리 집계
- 카테고리별 수주 (소파TF 관련 데이터 별도 추출)
- 전월 데이터가 있으면 전월 대비 증감 계산
- 전년 동월 데이터가 있으면 YoY 증감 계산

**매출실적 분석:**
- 동일한 기준으로 분석

**목표 대비 분석:**
- 목표 파일이 있으면 달성률 계산
- 없으면 "목표 미설정" 표기

**외부활동 분석:**
- 활동별 참여 현황, 수주 연계 실적 집계

### STEP 3. 경쟁사 동향 수집

WebSearch로 최근 30일간 경쟁사 동향을 검색합니다.

**검색 키워드:**
- 한샘 프로모션, 한샘 신제품, 한샘 이벤트
- 리바트 프로모션, 리바트 신제품
- 까사미아 프로모션, 까사미아 신제품
- 에몬스 프로모션, 에몬스 신제품
- 가구 시장 트렌드 YYYY년 MM월

**기존 주간 모니터링 참조:**
- `20-operations/21-monitoring/` 에서 해당 월의 주간 모니터링 파일을 읽어서 월간 종합 요약에 활용

**날짜 검증 규칙:**
- 해당 연도 기사만 포함
- 날짜 불명확한 기사는 제외

### STEP 4. 보고서 조립

`00-system/01-templates/monthly-report-template.md` 템플릿을 기반으로 보고서를 작성합니다.

**작성 원칙:**
- **Executive Summary**: 핵심 숫자 3개 + 주요 이슈 3줄 (피라미드 원칙)
- **데이터 섹션**: 실제 수치로 테이블 채우기
- **경쟁사 섹션**: 검색 결과 + 주간 모니터링 종합
- **소파TF**: 소파 카테고리 데이터가 있을 때만 포함
- **액션 아이템**: So What / Now What 기법으로 도출

**숫자 표현 규칙:**
- 큰 금액은 억/만원 단위로 변환
- 증감률은 %로 표기
- 핵심 숫자는 3개 이하로 선별

### STEP 5. 로컬 저장

파일을 아래 경로에 저장합니다:
```
20-operations/25-monthly-reports/YYYY-MM_월간보고서.md
```

### STEP 6. Confluence 업로드

`/confluence-handler` 를 활용하여 Confluence에 업로드합니다.

**업로드 규칙:**
- 스페이스: 리테일사업팀 스페이스
- 상위 페이지: "월간 보고서" (없으면 생성)
- 페이지 제목: `[YYYY년 MM월] 리테일사업팀 월간 보고서`
- 기존 페이지가 있으면 업데이트, 없으면 신규 생성

### STEP 7. 완료 보고

사용자에게 결과를 요약해서 보여줍니다:

```
✅ 2026년 3월 월간 보고서 완성

📂 로컬: 20-operations/25-monthly-reports/2026-03_월간보고서.md
🌐 Confluence: [페이지 URL]

📊 핵심 수치
- 수주: OO억원 (목표 대비 OO%, 전월비 +OO%)
- 매출: OO억원 (목표 대비 OO%, 전년비 +OO%)

⚠️ 참고사항
- [건너뛴 섹션이 있으면 안내]
- [데이터 이상치가 있으면 안내]
```

## 주의사항

- 엑셀 파일 컬럼명이 다를 수 있으므로, 먼저 시트/컬럼 구조를 확인 후 분석
- 전월/전년 데이터가 없으면 해당 비교 항목은 "데이터 없음" 표기
- 경쟁사 검색 결과가 없으면 "특이사항 없음"으로 표기
- Confluence 업로드 실패 시 로컬 파일만 저장하고 안내
- 보고서 생성 후 사용자에게 검토 요청 (자동 발송하지 않음)
